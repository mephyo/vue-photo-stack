# vue-photo-stack

>A gallery album icon component for Vue.js.

![Example](https://github.com/mephyo/vue-photo-stack/raw/master/docs/sample.png)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```
